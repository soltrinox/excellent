Type Definitions

CtrlName

It is a standard JavaScript nested-name string, made up by 1 or more JSName strings, joined by a dot:

module_1.$name2._ctrl3

It cannot start or end with a dot. And it cannot have any spaces in between.

It represents a full controller name. And if the name contains more than one JSName, then the first name always refers to a module, followed by either a simple or nested controller name implemented in that module.

Type:
  • string

JSName

It is a string that complies with the open-name syntax for JavaScript variables:

  • It must contain 1 or more symbols
  • It is treated as case-sensitive
  • Allowed symbols are: a-z, A-Z, 0-9, $ and _
  • It cannot start with a digit (0-9)
Type:
  • string
